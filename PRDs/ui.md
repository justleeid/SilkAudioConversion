## 1. 设计理念

### 1.1 设计哲学

**"少即是多，但少必须足够"**

高级感 = 克制 × 一致 × 细节

### 1.2 设计原则

| 原则 | 描述 | 实践方式 |
|------|------|----------|
| 克制 | 不做多余装饰 | 减少视觉噪音，留白呼吸 |
| 一致 | 全平台视觉语言统一 | 设计系统组件化 |
| 精致 | 细节处体现品质感 | 微动效、圆角、阴影 |
| 流畅 | 交互反馈自然顺滑 | 60fps 动效，即时响应 |
| 清晰 | 信息层次分明 | 字体层级、色彩对比 |

### 1.3 设计灵感参考

| 参考 | 借鉴点 |
|------|--------|
| Linear | 极简主义、微动效、深色模式 |
| Vercel | 几何感、单色为主、精致细节 |
| Apple | 留白、字体排印、流畅动效 |
| Raycast | 效率工具感、键盘优先、简洁 |

### 2.2 色彩变量定义

```scss
:root {
  // 主色
  --color-primary: #0066FF;
  --color-primary-hover: #0052CC;
  --color-primary-active: #0040A0;
  
  // 功能色
  --color-success: #10B981;
  --color-warning: #F59E0B;
  --color-error: #EF4444;
  --color-accent: #00C896;
  
  // 阴影
  --shadow-sm: 0 1px 2px rgba(0, 0, 0, 0.05);
  --shadow-md: 0 4px 6px rgba(0, 0, 0, 0.07);
  --shadow-lg: 0 10px 15px rgba(0, 0, 0, 0.1);
}
```

### 2.3 色彩使用比例（60-30-10 法则）

- **60%** 主背景色
- **30%** 次要色/表面
- **10%** 强调色（仅用于主要按钮、进度条、选中状态、重要通知）

## 6. 动效规范

### 6.1 动效时间曲线

| 场景 | 时长 | 曲线 |
|------|------|------|
| 快速反馈 | 150ms | cubic-bezier(0.4, 0, 0.2, 1) |
| 标准动效 | 200ms | cubic-bezier(0.4, 0, 0.2, 1) |

### 6.2 动效场景

| 场景 | 动效 | 时长 |
|------|------|------|
| 按钮悬停 | 背景色过渡 | 150ms |

| 卡片悬停 | 阴影 + 位移 | 200ms |
| 文件上传 | 淡入 + 上滑 | 300ms |
| 转换完成 | 弹性缩放 | 400ms |
| 进度条 | 线性增长 | 持续 |

## 9. 深色模式

### 9.1 深色模式色彩

- 避免纯黑，使用深灰 #0F1115
- 降低对比度，避免刺眼
- 阴影更深，增强层次感
- 强调色降低饱和度

### 9.2 深色模式适配

- 自动检测系统主题偏好
- 提供手动开关选项
- 保证 WCAG AA 对比度标准

## 10. 可访问性 (A11y)

### 10.1 键盘导航

- 所有可交互元素支持 Tab 键导航
- Enter 键触发按钮
- Escape 键关闭对话框
- 箭头键操作列表

### 10.2 屏幕阅读器

- 为图标添加 aria-label
- 表单添加关联标签
- 列表项添加角色定义
- 动态内容使用 aria-live

### 10.3 对比度标准

- 正文：WCAG AA 级 (4.5:1)
- 图形元素：WCAG AA 级 (3:1)
